# Open Civil Data Portal

نسخة نظيفة تجريبية لمنصة النسيج المدني.

## المكونات الحالية

- واجهة ثابتة قابلة للنشر عبر GitHub Pages.
- GitHub Actions لنشر مجلد `deploy-static`.
- هدية مجانية للزوار: إشعار تفاعلي يمكّن الزائر من:
  - إدخال اسم المؤسسة أو الشخص.
  - إدخال اسم الرابط أو الخدمة.
  - مسح الرابط.
  - إنشاء بطاقة QR.
  - تحميل البطاقة أو نسخ كود QR.

## لاحقًا

- ربط Supabase للبيانات والـ API.
- الحفاظ على نفس فلسفة المشروع: واجهة عامة للزوار، وخدمات عملية، ثم لوحة إدارة لاحقًا.
